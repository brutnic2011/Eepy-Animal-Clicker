Requisitos:
Tener instalado Python (versión 3.6 o superior).
Instalar las librerías con el comando: pip install pgzero y pip install pygame.
Cómo ejecutar el juego:
Opción A (Thonny): Simplemente abre el archivo .py en Thonny y presiona el botón Run. (Asegúrate de que Thonny esté en "Modo Pygame Zero" en el menú Visualización).
Opción B (Terminal/VS Code): Ejecuta el comando pgzrun Eepy Animal Clicker.py en la terminal.
Estructura de archivos:
Recuerda que la carpeta images debe estar en la misma carpeta que tu código para que el juego funcione correctamente
